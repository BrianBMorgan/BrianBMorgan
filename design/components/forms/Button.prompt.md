Primary call-to-action; use for the main action in a section, gold-primary for the single most important action per view.

```jsx
<Button variant="primary" size="lg" iconRight={<Icon name="arrow-right" size={18} />}>View work</Button>
<Button variant="secondary">Download CV</Button>
<Button variant="link">Read more</Button>
```

Variants: primary (gold), secondary (paper + hairline), ghost, inverse (charcoal), link. Sizes: sm/md/lg. Pass `as="a"` + `href` for link buttons.
