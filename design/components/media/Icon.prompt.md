Lucide icon wrapper — the ONLY icon set in this system. Requires the Lucide UMD script on the page. Never hand-roll SVG icons.

```jsx
<Icon name="arrow-up-right" size={18} strokeWidth={1.75} />
```

Load once per page: `<script src="https://unpkg.com/lucide@latest"></script>` then the wrapper hydrates itself.
