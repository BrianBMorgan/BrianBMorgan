When Foundry shipped the platform, they wanted the launch to feel like the product: considered, unhurried, and honest about what works. One stage, 300 seats, no confetti.

## The work

I owned the narrative arc and the run of show — a 90-minute program built around one live demo rather than a feature list. We rehearsed the demo to be failure-proof: local fallbacks, staged data, and a kill-switch path the audience would never notice.

Stage design borrowed the product's own design system; the slides and the software were visibly the same thing. That coherence was the message.

## What endured

- The demo ran live without a fallback triggered.
- Recordings of the program drove sign-ups for a quarter — the event kept working after the room emptied.
