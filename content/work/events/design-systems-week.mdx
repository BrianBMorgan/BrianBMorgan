The organizers asked for a closing talk that would send 400 practitioners home with something sturdier than inspiration. The result was "Tokens Before Taste" — an argument for sequencing system decisions so brands survive their own redesigns.

## The work

The keynote ran 40 minutes with live examples pulled from three shipped systems. The next morning's workshop took 32 people through building a token architecture from a blank file to a themed component — every attendee left with a working repo, not a slide deck.
